# 💫 About Me:
Suraj Laxman Jadhav is a Java Full Stack Developer with a background in Electronics and Telecommunication Engineering. He has experience working with Java, Spring Boot, Hibernate, MySQL, Angular, REST APIs, Microservices, Spring Security, API Gateway, and JWT authentication. He has developed several projects including a Books Management System, Student Attendance Management REST API, Final IPL MVC project, and Track Information APIs for a music streaming platform. His final year project was published at an IEEE event. Along with software development, he is also preparing for UPSC Prelims 2025 and CDAC C-CAT, with strong interest in DSA, LeetCode problem solving, and interview preparation for Java, JavaScript, and TypeScript. He served as the Sports Head in his college NSS club and is continuously improving his technical, leadership, and problem-solving skills while aiming to build advanced AI-enabled microservices projects like Smart Recruiter and Smart Hospital Management platforms.
